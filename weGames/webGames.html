Frogger


Play Frogger!
Pass Through the chaotic streets as a frog. You only have a limited life time of 3. Take your chances to reach to the finish line.

Created by an array of cars and a moving image of a frog. Once the frog is fully hit by the Car you have lost a life. You only have 3 lives until it asks you to play again.

Tic-Tac-Toe


Play Tic-Tac-Toe
The Classic Game of Tic-Tac-Toe. Go against the computer and try to win as much as possible.

Created by a a Grid Container and an Array to hold the values. Uses a simple if else statements to check if there is a winner or a loser. If there is no winner or loser by 9 moves then it is a tie.

Breakout


Play 2048
Simple Breakout Game. Break the blocks to get through and don't let the ball pass the paddle or you will lose!

Created by an array of class Block, a moving ball, and a paddle. The ball is the only constant moving object and the paddle will only move when the user presses left or right arrows.

2048


Play 2048
Add as much cells of the same kind to reach the highest cell of 2048! And don't fill up the board with random cells.

Created by a 2d array and a grid container. Uses for loops and if else statements to check if there is cells of the same kind or if there is an empty space that is available to move into. Once the board is filled it checks if there are any cells to be paired, if there are none GameOver.
